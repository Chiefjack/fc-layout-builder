Instructions:

'Drag and Drop' the units from the box in the centre to the boxes on the left.
To remove a unit from a box on the left, 'Drag and Drop' the unit into the bin in the centre.
Once you are happy with your layout, please screenshot it and send it in #outcasts-layouts in the Frontier Chronicles Discord server.

Known Issues:

You can 'Drag and Drop' a unit from the centre list directly into the bin, causing it to be removed from the centre list.
It isn't necessarily 'mobile optimised', however you should still be able to create layouts on mobile.
'Drag and Drop' doesn't work on some browsers (mostly outdated browsers).
It has been tested and works correctly on:
- Google Chrome for Windows
- Microsoft Edge for Windows
- Firefox for Mac
- Safari for iPadOS
- Safari for iOS
- Chrome for iOS
- Edge for iOS
- Firefox for iOS
- Android Default Browser
It has been tested and doesn't work on:
- Internet Explorer
